Sobre Mim

Sou uma pessoa tranquila que adora encontrar soluções para os desafios do dia-a-dia.

Graduado em engenharia elétrica e como fiz ensino médio numa escola técnica, me formei em dois cursos técnicos: Mecatrônica e Eletrotécnica. Com uma formação técnica, desenvolvi um pensamento crítico. Eu analiso as situações com cuidado para tomar uma decisão justa e coerente com o problema. Na minha trajetória profissional, eu tenho experiência com manutenção predial e elaboração de orçamentos para obras em edifícios de missão crítica. Atuando nessa área a minha comunicação foi essencial para ter um relacionamento saudável com o cliente e transmitir de maneira ágil e assertiva as informações necessárias para a solução das adversidades.

Atualmente estou em transição de carreira, sempre tive afinidade com tecnologia. Já trabalhei utilizando a Linguagem Ladder para programação de PLC e o trabalho de conclusão do meu curso de engenharia foi o desenvolvimento de uma aplicação para o controle de geradores elétricos utilizando MatLab e LabView. Realizei o bootcamp da Generation de Desenvolvedor Java, onde estudei a linguagem Java, fundamentos de banco de dados e persistir dados com MySQL, aprendi a criar um API RESTful com Spring e consumido pelo framework Angular, e realizei o deploy do back-end e front-end separadamente.
